# WaterWakey

A device that shoots water at you to wake you up at a preset time.  


## Components  
- Arduino uno  
- Bottle  
- Connection Wire  
- Mini water pump 4-12v  
- DS1307 AT24C32 IIC RTC module  
- Battery
